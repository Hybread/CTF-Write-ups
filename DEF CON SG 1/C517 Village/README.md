Each player was required to solve 1 of each "levels" of both categories to achieve a stamp on the respective level slot.

For Reverse, Anchor 1's level 3 was locked behind the first 3 challenges, so I had to solve the first 3 Level 2's before being able to attempt the Level 3 challenge.

For Air-Gapped, the first level was just an MCQ + QnA of the given Cross-Domain Solution (CDS) infrastructure simulation, while the other two was exploiting a known Ghostscript CVE to achieve RCE, which was used by the CDS infra itself.
