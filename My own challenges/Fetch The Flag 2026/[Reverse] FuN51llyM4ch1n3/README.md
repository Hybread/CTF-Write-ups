## Introduction
FuN51llyM4ch1n3 is a FSM (Finite State Machine) crackme challenge inspired by one of the challenges in Google Mandiant's Flare-On CTF 2025, featuring a graph problem inside of a FSM that carried 96,767 unique states. 
Very thankful to have had the amazing opportunity to create this challenge for Snyk's annual Fetch The Flag CTF (2026).
