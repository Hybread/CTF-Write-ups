# CYNX 1103

## Introduction
CYNX 1103 is a medium cryptography challenge which challenges the problem solving ability of players. The title is a direct flip of the famous `Cicada 3301` which is also a hint to the challenge itself. 
The player must identify the Gematria Primus encoded runes which include a Vigenere layer as well as a Shift-cipher layer to retrieve the keys needed to obtain the flag.
