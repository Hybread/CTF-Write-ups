# N0mn0M.exe

## Introduction
N0mn0M is a multi-layer reverse engineering challenge which involves multiple knowledge levels of packing. This includes UPX packing, custom UPX packing, and custom algorithmic packing of a binary. 
The user will have to reverse the given binary and the packing methods it includes, to reveal the real encryptor binary which contains the encryption methods to decrypt the flag. 

*No official writeup was prepared for this challenge, but during the competition, 1 player managed to solve it with IDA MCP, then proceeded to manually solve it after the competition.
As such, I will be posting the writeup in this folder. Enjoy!
